# Go-To-Market Wedges (Issue #17)

This document tracks marketing and sales tasks for three GTM wedges identified
in roadmap:

1. **Windows Refugee** – target game studios and .NET enterprises migrating
   off GitHub-hosted runners.
2. **Blacksmith Upgrade** – highlight compliance and performance advantages
   versus Blacksmith.
3. **Buildkite Displacement** – enable Buildkite users on GitHub to migrate to
   RunnerCloud.

## Executive Plan

- **Windows Refugee**
  - Publish blog post announcing Windows Server 2025 support and GPU-backed
    builds at $0.002/min.
  - Run paid ads on game development forums and Reddit.
  - Create case study with Unity game studio partner.

- **Blacksmith Upgrade**
  - Develop migration guide and table comparing Cloud vs Blacksmith.
  - Host webinar focusing on compliance, SOC 2, FedRAMP.
  - Offer 30-day trial for existing Blacksmith customers.

- **Buildkite Displacement**
  - Write analysis showing cost savings and removal of per-seat tax.
  - Publish ROI calculator (linked from TCO calculator).
  - Provide scripts to import Buildkite pipelines into RunnerCloud.
