## 📋 WHAT EACH WORKFLOW DOES

**Realtime Branch Monitor** (Every 5 min)
- Validates branch naming instantly
- Detects stale branches
- Scans metadata for accidental secrets

**Intelligent Cleanup** (Daily 2 AM)
- ML-aware stale detection
- Archives branches before deletion
- Recovery available 30 days

**Commit Validation** (On push)
- Prevents non-conventional commits
- Ensures ticket linking
- Blocks at pre-push hook

**PR Quality Gates** (On PR open)
- Requires description + linked issue
- Enforces labels
- Blocks merge if criteria unmet

**Advanced Secret Scanning** (24/7)
- TruffleHog + GitLeaks + entropy detection
- Scans entire git history
- Auto-blocks commits with secrets
- Notifies immediately

**Commit Signing** (On merge)
- Ensures all commits to main are GPG/SSH signed
- Verifies contributor identity
- Blocks merge if unsigned

**Code Owner Router** (On PR creation)
- Auto-assigns reviewers based on CODEOWNERS
- Routes to experts, not just anyone
- Prevents bottlenecks

**Review SLA** (Every 30 mins)
- Hotfix: 30 min SLA (or escalate)
- Critical: 2 hour SLA
- Standard: 24 hour SLA
- Tracks and escalates violations

**Compliance Report** (Daily 8 AM)
- Scorecard: 950+/1000 target
- Metrics: All compliance dimensions
- Delivered: Email + Slack + GitHub

**Commit Scoring** (Per-commit)
- Quality metrics: Message, signing, scope, tests
- 0-100 score per commit
- Hall of fame for 100-scorers
