   - Schedule: Daily 2 AM UTC
   - Features: ML-aware stale detection, archive branches
   - Result: No manual cleanup needed

3. **commit-validation.yml**
   - Trigger: On PR creation (runs automatically)
   - Checks: Conventional format, ticket linking, message length
   - Action: Blocks non-compliant commits

4. **pr-quality-gates.yml**
   - Trigger: On PR open/sync
   - Checks: Description required, labels, reviewers, not draft
   - Action: Auto-blocks if criteria unmet

   - Trigger: 24/7 (push + PR)
   - Checks: TruffleHog, GitLeaks, entropy detection, history
   - Action: Auto-block + alert + remediate

   - Trigger: Before merge
   - Checks: GPG/SSH signature verification
   - Action: Block merge if unsigned

7. **codeowner-router.yml**
   - Trigger: On PR creation
   - Checks: Extract CODEOWNERS, route to experts
   - Action: Auto-assign reviewers

8. **pr-review-sla.yml**
   - Trigger: Every 30 minutes
   - Checks: PR age vs SLA (30min/2hr/24hr)
   - Action: Escalate if violated

   - Trigger: Daily 8 AM UTC
