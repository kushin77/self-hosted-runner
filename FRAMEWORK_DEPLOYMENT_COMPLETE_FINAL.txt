HOW TO COMPLETE GITHUB SYNC (Final step)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**Step 1: Create GitHub Personal Access Token (PAT)**
   1. Go to: https://github.com/settings/tokens
   2. Click "Generate new token (classic)"
   3. Scopes needed: ✅ repo (full control of private/public repos)
   4. Create and copy token (save securely)

**Step 2: Export token in your terminal**
   export auth_token="GITHUB_TOKEN_HERE"
   
   Verify:
   curl -i -H "Authorization: token $auth_token" https://api.github.com/user
   (Should return HTTP/2 200 with your user info)

**Step 3: Run sync scripts**
   Option A: Full automation
     bash scripts/github/sync-github-deployment.sh --all
   
   Option B: Step by step
     bash scripts/github/sync-branch-protection.sh
     bash scripts/github/sync-issues.sh --create
     bash scripts/github/sync-issues.sh --verify

**Step 4: Verify on GitHub**
   • Branch protection: https://github.com/kushin77/self-hosted-runner/settings/branches
   • Issues: https://github.com/kushin77/self-hosted-runner/issues
