1. Create database password secret (example):

   kubectl create secret generic harbor-db-password --from-literal=password="YOUR_DB_PASSWORD"

2. Install chart (example - local Helm chart):

   helm install my-harbor ./deploy/harbor \
     --set persistence.enabled=true \
     --set tls.enabled=false

3. Production guidance:
   - Use an external Postgres DB and provide `database.host` and `database.passwordSecret`.
   - Configure Redis and Trivy as needed.
   - Enable TLS by creating `harbor-tls` secret and setting `tls.enabled=true`.
